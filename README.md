## 📡 这是什么

StockHeat 是一个**实时股票板块热度监控工具**，通过聚合抖音、微博、小红书等社交平台的舆论数据，结合东方财富实时行情，计算每个板块ETF的综合热度分，并生成排行榜和买卖信号。

**核心理念**：社交媒体上的讨论热度往往是市场情绪的前瞻指标。当某个板块在抖音微博上突然爆火时，往往意味着资金即将涌入或正在涌入。

## ✨ 功能

- 🔥 **综合热度排行** — 20个板块ETF实时排名，0-100热度分
- 📱 **多平台聚合** — 抖音热搜 + 微博热搜 实时抓取，小红书智能模拟
- 📊 **真实行情** — 上证/深证/创业板/科创50 指数 + 20个ETF实时价格
- 👩‍👧 **宝妈指数** — 追踪宝妈炒股圈热门板块（ETF定投/高股息/白酒/医药/可转债等10个方向）
- 🎯 **交易信号** — 强烈买入 / 建议关注 / 持有观察 / 建议回避 自动生成
- 🔍 **关键词匹配** — 200+ 行业关键词自动匹配社交话题到对应板块
- 📈 **情绪趋势** — 24小时全市场情绪走势图
- 🔴🟢 **数据溯源** — 每个板块标注数据真伪（绿色=实时、橙色=模拟）
- ⚡ **30秒自动刷新** — 无需手动操作

## 📊 热度计算逻辑

```
综合热度分 = 讨论量(28%) + 情绪倾向(42%) + 行情动量(30%)

讨论量：各平台讨论数取对数标准化到0-100
情绪倾向：看多关键词(+分) vs 看空关键词(-分)
行情动量：ETF实际涨跌幅映射到0-100
```

**交易信号阈值：**
- ≥78分 + 上涨 + 看多 → 🔥 强烈买入
- ≥65分 + 跌幅<1% + 偏多 → 📊 建议关注
- ≥48分 + 偏多 → 👀 持有观察
- <38分 或 大跌+看空 → ⚠️ 建议回避
